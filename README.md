# Kicko Web

Kicko Web is a modern and high-performance technology platform developed using React, Vite, TypeScript, and Tailwind CSS. The project focuses on delivering responsive web experiences, scalable architecture, and modern UI design for businesses and startups.

---

## 🚀 Features

- Modern Responsive UI
- Fast Performance with Vite
- React + TypeScript Architecture
- Tailwind CSS Styling
- Reusable Components
- Mobile Friendly Design
- Smooth Animations
- SEO Optimized Structure
- Scalable Project Setup

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Ajaykicko/kicko_web.git
```

Move into the project folder:

```bash
cd kicko_web
```

Install dependencies:

```bash
npm install
```

---

## ⚙️ Environment Setup

Create a `.env` file in the root directory and add your API key:

```env
GEMINI_API_KEY=your_api_key_here
```

Get your API key from:

https://aistudio.google.com/app/apikey

---

## ▶️ Run Development Server

```bash
npm run dev
```

The application will run at:

```bash
http://localhost:3000
```
